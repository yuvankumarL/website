target Specifies where to open the linked document.

Common values:
            _self: Opens in the same window (default).
            _blank: Opens in a new window or tab.
            _parent: Opens in the parent frame.
            _top: Opens in the full body of the window, breaking out of frames.

rel (Relationship):
Defines the relationship between the current document and the linked document.
Common values:
        nofollow: Instructs search engines not to follow the link.
        noopener: Prevents the new page from accessing the window.opener object.
        noreferrer: Prevents sending referrer information to the new page.
